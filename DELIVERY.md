# Delivery

Este workspace ya está preparado como base de entrega del proyecto.

## Qué incluye el paquete

- `README.md`: portada principal del proyecto
- `notia/`: runtime y demo principal
- `notia-core/`: núcleo semántico
- `iota-identity-backend/`: backend de identidad

## Qué no se empaqueta

- `.env`
- credenciales, JWTs y strongholds
- `node_modules/`
- `dist/`
- `target/`
- artefactos temporales y de testing local

## Qué te queda hacer a ti

- rellenar links finales de vídeo, explorer y submission
- configurar secretos en la máquina final
- probar una vez el flujo completo de demo
- enviar el formulario del hackathon

## Comando principal de demo

```bash
cd notia
./bin/notia
```
